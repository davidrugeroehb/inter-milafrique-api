Inter Milafrique API

API voor mijn zaalvoetbalploeg

localhost URL:

http://localhost:3000/api

Nieuws Endpoints

GET /api/nieuws

Haalt alle nieuwsartikelen op

GET /api/nieuws/:id

Haalt één nieuwsartikel op

POST /api/nieuws

{
  "title": "Grote overwinning",
  "content": "Inter Milafrique won met 5-2",
  "user_id": 1
}

PUT /api/nieuws/:id

Update een nieuwsartikel

DELETE /api/nieuws/:id

Verwijdert een nieuwsartikel

Spelers Endpoints

GET /api/spelers

Haalt alle spelers op

GET /api/spelers/:id

Haalt één speler op ID

POST /api/spelers

{
  "name": "David Rugero",
  "email": "davidrugero@intermilafrique.be",
  "position": "Middenvelder"
}

PUT /api/spelers/:id

Update een speler

DELETE /api/spelers/:id

Verwijdert een speler

Antwoorden

Success

{
  "success": true,
  "data": {}
}

Error

{
  "success": false,
  "message": "Error"
}

HTTP Status

  • 200 – OK
  • 201 – Created
  • 400 – Bad Request
  • 404 – Not Found
  • 500 – Server Error
