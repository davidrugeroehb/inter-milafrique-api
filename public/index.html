Inter Milafrique API

API voor mijn zaalvoetbalploeg

localhost URL:

http://localhost:3000/api

Nieuws Endpoints

GET /api/nieuws

Haalt alle nieuwsartikelen op

GET /api/nieuws/:id

Haalt één nieuwsartikel op

POST /api/nieuws

{
  "title": "Grote overwinning",
  "content": "Inter Milafrique won met 5-2",
  "user_id": 1
}

PUT /api/nieuws/:id

Update een nieuwsartikel

DELETE /api/nieuws/:id

Verwijdert een nieuwsartikel
